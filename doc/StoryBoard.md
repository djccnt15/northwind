**목적**: 미인증 사용자에게 서비스를 소개하고 로그인/회원가입으로 유도

```
┌──────────────────────────────────────────────────────────────────┐
│ Welcome to Northwind!                                            │
├──────────────────────────────────────────────────────────────────┤
│                                                                  │
│  Northwind is a sample ERP application built with                │
│  React and Spring.                                               │
│                                                                  │
│  Please [login] or [sign up] to continue.                        │
│                                                                  │
│                   Standing on the                                │
│                   Shoulders of Giants                            │
│                                                                  │
├──────────────────────────────────────────────────────────────────┤
│ © 2024 Northwind. All rights reserved.      [GitHub] [Blog]      │
└──────────────────────────────────────────────────────────────────┘
```

**사용자 흐름**:
- `login` 링크 클릭 → S-02 로그인
- `sign up` 링크 클릭 → S-03 회원가입
- 이미 로그인된 상태에서 접근 시 `/home`으로 리다이렉트

### S-02 로그인 ✅

**목적**: 시스템 사용자 인증

```
┌──────────────────────────────────┐
│         Please Sign In           │
├──────────────────────────────────┤
│  [ID                           ] │
│  [Password                     ] │
│                                  │
│  [       Login       ]           │
│                                  │
│  ☐ Remember Me   ☐ Remember ID  │
├──────────────────────────────────┤
│  Don't have an account?          │
│  Create one →                    │
└──────────────────────────────────┘
```

**기능**:
- ID/비밀번호 입력 후 로그인
- `Remember Me`: 세션 유지 (쿠키 기반 remember-me)
- `Remember ID`: 아이디 localStorage 저장
- 로그인 실패 시 에러 메시지 표시
- 로그인 성공 → 이전 페이지 또는 `/home`으로 이동

### S-03 회원가입 ✅

**목적**: 신규 사용자 계정 생성

```
┌──────────────────────────────────────┐
│           Please Sign Up             │
├──────────────────────────────────────┤
│  [ID                               ] │
│  [Email              ] [Check Email] │
│  [Password                         ] │
│  [Confirm Password                 ] │
│                                      │
│  [          Sign Up          ]       │
│   ※ 이메일 중복 확인 후 활성화        │
├──────────────────────────────────────┤
│  Already have an account?            │
│  Log in →                            │
└──────────────────────────────────────┘
```

**기능**:
- 이메일 중복 확인 (`Check Email`) 필수 — 확인 전 `Sign Up` 비활성화
- 비밀번호 불일치 시 경고
- 검증 오류(code 1400) → 오류 목록 모달 표시
- 가입 성공 → 성공 모달 → 로그인 페이지 이동 링크 제공

**언어 자동 설정 (가입 시)**:
- 프론트엔드가 브라우저 `navigator.language` (또는 `navigator.languages[0]`)를 읽어 회원가입 요청 body에 포함
- 백엔드는 해당 언어 코드가 `supported_lang` 테이블에 존재하면 그 값을 `preferred_lang_id`로 저장
- `supported_lang`에 없는 언어이면 `en`으로 등록된 레코드를 기본값으로 저장
- 이후 S-11 프로필 화면에서 언제든지 변경 가능

**모달 흐름**:
```
가입 성공                       검증 오류
┌───────────────────┐          ┌─────────────────────────────┐
│   Sign Up Success!│          │ Signup Failed           [X] │
│   Go to [Log in →]│          ├─────────────────────────────┤
└───────────────────┘          │ · 비밀번호는 8자 이상이어야...  │
                               │ · 아이디는 영문/숫자만...       │
                               └─────────────────────────────┘
```

### S-10 홈 대시보드 ✅ (플레이스홀더 → 🔲 확장 필요)

**목적**: 로그인 직후 진입점. 업무 연속성을 위한 MRU 및 알림 표시

**현재 상태**: 환영 메시지만 표시 (플레이스홀더)

**목표 화면**:
```
┌─────────────────────────────────────────────────────────────────┐
│ Welcome to Northwind! {username}                                 │
├───────────────────────────┬─────────────────────────────────────┤
│  최근 방문 (MRU)           │  알림                               │
│  ┌────────────────────┐   │  ┌───────────────────────────────┐  │
│  │ 거래처: ACME Corp  │   │  │ ⚠ 재고 부족: 상품 3건          │  │
│  │ 주문: ORD-2025-042 │   │  │ ⚠ 발주 승인 대기: 2건          │  │
│  │ 상품: Chai         │   │  └───────────────────────────────┘  │
│  └────────────────────┘   │                                     │
├───────────────────────────┴─────────────────────────────────────┤
│  요약 통계                                                        │
│  [이번달 매출]  [미결 주문]  [재고 부족 품목]  [미결 발주]          │
└─────────────────────────────────────────────────────────────────┘
```

**기능 (계획)**:
- 최근 방문 테이블/데이터 (`mru`) 표시 — 클릭 시 해당 상세 페이지 이동
- 재고 부족 알림 (`reorder_level` 기반)
- 발주 승인 대기 알림
- 월별 매출/매입 요약 통계 카드

### S-20 사용자 관리 ✅ (ADMIN)

**목적**: 전체 사용자 계정의 조회·수정·권한 관리

```
┌──────────────────────────────────────────────────────────────────┐
│ Admin - User Management                                          │
├──────────────────────────────────────────────────────────────────┤
│ [🔍 검색창          ]                                            │
├──┬────────┬────────┬──────────┬───────┬──────┬───────┬──────────┤
│  │Enabled │ID      │Email     │Title  │Team  │Role   │Live Until│
├──┼────────┼────────┼──────────┼───────┼──────┼───────┼──────────┤
│  │   ✓    │admin   │...@...   │Manager│Sales │[Roles]│2099-...  │
│  │   ✓    │user1   │...@...   │Staff  │Dev   │[Roles]│2099-...  │
│  │   ✗    │user2   │...@...   │-      │-     │[Roles]│-         │
├──┴────────┴────────┴──────────┴───────┴──────┴───────┴──────────┤
│                                        [< 1 2 3 ... >] 10/page  │
└──────────────────────────────────────────────────────────────────┘
```

**기능**:
- 인라인 행 편집 (Enabled, ID, Email, Title, Team, Live Until)
- 빠른 검색 (키워드 필터)
- `[Roles]` 버튼 → 권한 관리 모달
- `[Reset]` 버튼 → 비밀번호 초기화 (confirm 다이얼로그)
- 서버 사이드 페이지네이션

**권한 관리 모달**:
```
┌──────────────────────────────────┐
│ Role Management   [Save][Cancel] │
├──────────────────────────────────┤
│  ☑ ADMIN                        │
│  ☑ USER                         │
│  ☐ MANAGER                      │
└──────────────────────────────────┘
```
